The notebook uses SpliceViT-based style transfer for generating new images
Our Goal Generate style-transferred images that preserve facial structure while modifying appearance.
What this notebook does
- Loads predefined (content, style) image pairs
-Runs SpliceViT to generate stylized images
Saves:
- Generated images
- Metadata logs (JSON / JSONL)
Partial runs and continuation are controlled within the code.
